🏥 Clinic Management System

A role-based Clinic Management System developed using ASP.NET Core MVC, SQL Server, ADO.NET, and Stored Procedures to streamline clinic operations, patient management, pharmacy workflows, laboratory processes, and appointment handling.

📌 Overview

The Clinic Management System is a multi-user healthcare application designed to improve the efficiency of clinic operations through role-based access and workflow automation.

The system enables healthcare staff to manage patient records, appointments, prescriptions, laboratory requests, medicine inventory, billing, and notifications within a centralized platform.
